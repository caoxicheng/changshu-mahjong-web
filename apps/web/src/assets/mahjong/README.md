将麻将牌图片资源放到这个目录，前端会自动优先使用图片。

推荐命名：

- `wan-1.png` 到 `wan-9.png`
- `tong-1.png` 到 `tong-9.png`
- `tiao-1.png` 到 `tiao-9.png`
- `wind-east.png`
- `wind-south.png`
- `wind-west.png`
- `wind-north.png`
- `dragon-red.png`
- `dragon-green.png`
- `dragon-white.png`

也兼容使用共享牌编码作为文件名：

- `wan-1.png`
- `tong-9.png`
- `wind-1.png`
- `dragon-3.png`

支持扩展名：`png`、`svg`、`webp`、`jpg`、`jpeg`
